# ACE-NET

ACE NET é um painel web auxiliar para configuração de portas em switches Cisco e Huawei.

## Funcionalidades

- Configuração de VLAN por porta (Access)
- Shutdown e No Shutdown
- Simulação antes da execução real
- Confirmação antes da aplicação em produção
- Modo Laboratório e Produção
- Interface visual com mapa de portas

## Exemplo

porta 10 vlan 20  
porta 5 shutdown  
porta 5 no shutdown  

Projeto em evolução.
